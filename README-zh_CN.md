# Obsidian 标题序号插件

一个强大的 Obsidian 插件，可自动为 Markdown 标题添加可自定义的序号。

## 功能特性

- **中文编码模式**：一键启用中文文档编号格式
  - H1: 一、
  - H2: (一)、
  - H3: 1.
  - H4: (1).
  - H5: ①.
  - H6: A.

- **多种编号样式**：支持 12+ 种不同的编号样式
  - 阿拉伯数字 (1, 2, 3...)
  - 中文小写 (一, 二, 三...)
  - 中文大写 (壹, 贰, 叁...)
  - 中文括号 ((一), (二), (三)...)
  - 中文顿号 (一、, 二、, 三、...)
  - 阿拉伯句号 (1., 2., 3....)
  - 阿拉伯括号 ((1), (2), (3)...)
  - 带圈数字 (①, ②, ③...)
  - 罗马数字 (I, II, III... / i, ii, iii...)
  - 字母 (A, B, C... / a, b, c...)

- **灵活的级别配置**：选择要应用编号的标题级别 (H1-H6)

- **每个级别独立样式**：为每个标题级别设置不同的编号样式

- **自定义分隔符**：配置数字之间的分隔符（例如 `.`, `-`, `/`）

- **独立编号模式**：每个标题级别独立编号，不带父级前缀

- **实时预览**：配置时实时查看序号效果

## 命令

| 命令 | 描述 |
|------|------|
| `添加标题序号` | 为当前文档的所有标题添加序号 |
| `清除标题序号` | 清除当前文档中所有标题的序号 |

## 配置

通过 `设置 → 社区插件 → 标题序号插件` 访问插件设置。

### 设置选项

| 选项 | 描述 |
|------|------|
| **中文编码模式** | 一键启用中文文档编号格式 |
| **起始标题级别** | 从哪个标题级别开始添加序号 (H1-H6) |
| **结束标题级别** | 到哪个标题级别停止添加序号 (H1-H6) |
| **序号分隔符** | 各级序号之间的分隔符（默认：`.`） |
| **独立编号模式** | 每个标题级别独立编号，不带父级前缀 |
| **H1-H6 编号样式** | 为每个标题级别选择编号样式 |

## 示例

### 中文模式示例

启用中文编码模式后的效果：

```markdown
# 一、引言

## (一)、背景

### 1. 历史

### 2. 现状

## (二)、方法

### 1. 方案A

### 2. 方案B

# 二、结论
```

### 默认设置示例（阿拉伯数字，分隔符 `.`）

```markdown
# 1 引言

## 1.1 背景

### 1.1.1 历史

### 1.1.2 现状

## 1.2 方法

### 1.2.1 方案A

### 1.2.2 方案B

# 2 结论
```

### 混合样式示例（H1=罗马数字，H2=阿拉伯数字，H3=小写字母）

```markdown
# I 引言

## I.1 背景

### I.1.a 历史

### I.1.b 现状

## I.2 方法

### I.2.a 方案A

### I.2.b 方案B

# II 结论
```

## 安装

### 从 Obsidian 社区插件安装

1. 打开 Obsidian 设置
2. 进入社区插件并关闭安全模式
3. 点击浏览并搜索 "Title Serial Number"
4. 安装并启用插件

### 手动安装

1. 从 [GitHub Releases](https://github.com/superjackvip/obsidian-title-serial-number-plugin/releases) 下载最新版本
2. 解压文件到你的仓库的 `.obsidian/plugins/obsidian-title-serial-number-plugin/` 文件夹
3. 重新加载 Obsidian 并在社区插件设置中启用插件
