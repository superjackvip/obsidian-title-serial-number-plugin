# CHANGELOG 更新日志

## [1.0.0] - 2024-04-13 (中文定制版)

### 🌐 本地化

- **README 汉化**：完整的中文文档 (`README-zh_CN.md`)
- **英文 README 更新**：添加中文定制版说明和链接
- **插件描述汉化**：manifest.json 和 package.json 已更新

### 🔧 修改优化

- **作者信息更新**：更新为 `superjackvip`
- **编译优化**：重新编译的 `main.js`

### ✨ 新增功能

- **中文编码模式**：一键启用中文文档编号格式
  - H1: 一、
  - H2: (一)、
  - H3: 1.
  - H4: (1).
  - H5: ①.
  - H6: A.

- **独立编号模式**：每个标题级别独立编号，不带父级前缀

- **更多编号样式**：
  - 中文括号 ((一), (二), (三)...)
  - 中文顿号 (一、, 二、, 三、...)
  - 带圈数字 (①, ②, ③...)

### 📋 完整功能列表

- 多种编号样式：支持 12+ 种不同的编号样式
- 灵活的级别配置：选择要应用编号的标题级别 (H1-H6)
- 每个级别独立样式：为每个标题级别设置不同的编号样式
- 自定义分隔符：配置数字之间的分隔符
- 实时预览：配置时实时查看序号效果
